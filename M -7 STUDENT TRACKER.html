Feature 01
🤝
Centralised Stakeholder Collaboration
One platform connecting Class, Teacher, Student & Parent — eliminating silos, enabling shared visibility.
  • Role-based dashboards for each stakeholder
  • Real-time data accessible by all roles
  • Shared student records in single source of truth
  • Announcements, messages & reports via one system
✅ Live in BIT CODE LAB
Feature 02
🤖
AI-Powered Improvement Tips (NEW)
Gemini API generates personalised study tips, predicts at-risk students, and auto-suggests intervention strategies.
  • Personalised tips per student based on grade patterns
  • Attendance drop prediction before it happens
  • Subject weak-spot detection across entire class
  • Auto parent notification for at-risk students
✨ Gemini API Powered
Feature 03
📜
Certificate AI Scanner & Vault (NEW)
Students upload certificate images — AI auto-extracts all information and verifies authenticity. Teacher controls final access.
  • Auto-extract: event, org, date, participant name
  • Fake certificate detection via pattern recognition
  • Students submit proof; teachers verify & approve
  • Complete vault: project, workshop, course, hackathon
🔍 AI Verified
Feature 04
📄
Auto Resume Builder at Graduation (NEW)
At Year 4 graduation, one click generates a professionally formatted resume from all 4 years of student data.
  • CGPA, SGPA from all 8 semesters
  • Internship, placement, hackathon history
  • All verified certificates auto-included
  • Activities, councils, sports, workshops
📄 1-Click Resume
